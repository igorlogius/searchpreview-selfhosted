# Docker

## Setup

run `docker-setup.sh` or  `docker build . -t searchpreview-backend`

## Start

run `docker-start.sh` or `docker run -p7050:7050 searchpreview-backend`

(pass the -d option to docker run in the background)

# Development 

## Requirements

- node + npm (https://nodejs.org/)

## Setup 

npm install

## Start

npm start
